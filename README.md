# Antares Systems - ITAD Platform

## Overview
Antares Systems is a premium B2B IT Asset Disposition (ITAD) provider based in London. This platform showcases their secure data destruction services, IT recycling, and refurbished equipment sales.

## Tech Stack
- **Framework**: Astro 5 (Static Site)
- **Styling**: Tailwind CSS 4
- **Interactivity**: HTMX (Lead generation forms)
- **Forms**: Formspree Integration
- **Platform**: Cloudflare Pages

## Design Specs
- **Colors**:
  - Primary Blue: `#0081CC` (Bright Blue)
  - Success/Eco Green: `#10bb82` (Mountain Meadow)
  - Neutral Gray: `#9E9E9E`
- **Typography**: Inter (Primary), system-ui.

## Compliance & Trust
- **ICO Registration**: C18880232
- **Data Erasure Standards**: NIST 800-88, HMG IS5
- **Certifications**: NCSC-approved, GDPR compliant.

## Development
```bash
npm install
npm run dev
```

## Deployment
Automated via Cloudflare Pages on every push to `main`.
