# Project Context: Antares Systems

## Business Information
- **Company**: Antares Systems Ltd
- **Domain**: antaressystems.net
- **Business Model**: B2B ITAD, Data Destruction, Refurbished Sales.
- **Goal**: Establish a premium online presence for enterprise and public sector clients.

## Compliance Standards (Mandatory)
- **NIST 800-88**: Guidelines for Media Sanitization.
- **HMG IS5**: UK Government standard for data destruction.
- **NCSC**: Approved wiping and recycling methods.
- **GDPR**: Data protection compliance.

## Environment configuration
- **IDE**: Cursor / VS Code / Kilo Code
- **Agents**: Antigravity, Cline
- **MCP Servers Activated**:
  - `stitch`: UI/UX design and components.
  - `context7`: Library documentation (Astro, Tailwind, HTMX).
  - `github`: Repo management.
  - `brave-search`: Compliance and market research.
  - `filesystem`: File operations.

## Technology Decisions
- **Astro 5**: Chosen for speed and SEO.
- **HTMX**: Chosen for lightweight interactivity in lead forms without SPA complexity.
- **Cloudflare Pages**: Chosen for edge-global performance and easy CI/CD.
